# 계기

예전부터 생각하고 있었던 아이디어가 있었습니다.  
그건 바로 매일 나온 보카로 곡을 하나씩 넣은 달력을 만드는 것이었습니다.  
만약 아는 곡이 나온다면 '이 날엔 이런 곡이 나왔구나'하고 신기해하고, 모르는 곡이 나왔다면 '이건 무슨 곡일까'하면서 새로운 곡을 알아간다면 재미있겠다고 생각했습니다.

그렇게해서 든 생각을 실천하기 위해서 조금은 갑작스럽게 달력을 만들기 시작했고, 여러 사정이 있었지만 이렇게 온라인으로나마 달력을 만들게 되었습니다.

처음 만들어보는 프로젝트라 이런저런 부족한 점이 많지만 잘 부탁드립니다.

# 사용방법

위에 있는 달력의 월을 클릭하면 월별 달력을 보실 수 있습니다.  
그리고 월별 달력에서 날짜를 클릭하면 각 날짜에 무슨 곡이 나왔는지 볼 수 있습니다.  
각 페이지는 url로 들어갈 수도 있습니다.  
예를 들어, [2026/1](https://vocalendar.moe/2026/1)은 2026년 1월 달력으로, [2026/1/1](https://vocalendar.moe/2026/1/1)은 2026년 1월 1일 달력으로 이동합니다.
추가로, [2026/today](https://vocalendar.moe/2026/today) 링크로 들어가시면 2026년의 오늘의 어떤 곡이 나왔는지 바로 확인하실 수 있습니다.
